## I. 목적 · 전략
- P4는 KEESS 4대 필러 중 **콘텐츠 솔루션** 필러. 핵심 메시지 = "**기업의 모든 학습 니즈를, KG에듀원 기업교육**" — 8,426개 보유 콘텐츠와 파트너 네트워크를 **4개 축(보유·법정·제작·파트너)**으로 정리해 딜리버리.
- A안 정체성 = **4-Axis 구조**. 규모(보유)에서 시작해 법정 시리즈·자체 제작·파트너로 근거를 넓히고, 보유 축의 **대표 라인업 탐색기(AX·AI 34종 실데이터)**로 체험을 제공.
- 목표: 규모·체계·제작 역량 근거 + 탐색 체험을 제시해 **콘텐츠 구성/제작 문의 리드**를 인입.
- 범위 외: **과정리스트 다운로드 게이트(B안 전용, A안엔 없음)**. 가격·결제 미노출.

## III. 인터랙션 (시안 동작 보존, 홈 모션 규칙으로 통일)
- **Nav solid + 4축 스크롤스파이(axisnav)**: 스크롤 위치에 따라 현재 축 활성(IntersectionObserver rootMargin 시안값 준용, 홈 SubNav 스파이로 통일).
- **대표 라인업 탐색기(#axis-have · 핵심)**: 34종 실데이터.
  - 상태 `state`: `cat`(카테고리 탭)·`q`(검색, 160ms 디바운스)·`target`·`level`·`band`·`tool`·`sort`.
  - 탭(카테고리) + 검색 + 셀렉트 4(대상·레벨·기간·툴) + 정렬(기본·시간짧은·시간긴·레벨순) + count + reset(초기화).
  - 그리드 카드 렌더 → **카드 클릭/Enter/Space → 상세 모달(openDetail)**. 필터·검색·정렬 변경 시 즉시 재렌더.
- **상담 모달(consult)**: `data-consult` 트리거(hero·제작 문의·final 버튼) → openConsult(axis?) — axis 있으면 "문의 대상: {axis}" 컨텍스트 표기. 필드(cf-name·cf-org·cf-mail) 검증(이메일 정규식) → 성공(c-ok). 백엔드 없음.
- **상세 모달(detail)**: 과정 id로 상세(개요·목표·커리큘럼·툴·기간 등) 표시. ESC/스크림 닫기, 포커스 복원.
- **Reveal**: 홈 `useReveal`(th .16) 적용. reduced-motion 대응.

## IV. 비기능 요구
- **틴트**: 페이지 wrapper `.tint-p4`(surface#F7F4EF ink#15131D), `--maxw:1180px`. 필러색(#F58220 웜)은 포인트에만(eyebrow·강조·CTA·4축 맵 액센트).
- **색 스케일(데이터 시각화 예외)**: 4축 맵·탐색기 카테고리 배지·법정 타임라인 노드 등은 **--p4(웜) 및 보조 축색(법정=별도 톤)**을 포인트로만 허용. 시안의 --or/--or2/--rp/--rpsoft/--law 신설 대신 Design.md 토큰·`--p4` 파생으로 표현. 그 외 카드·배경·본문은 중립 베이스+틴트 유지.
- **폰트**: 전 페이지 통일 Pretendard Variable(self-host) 상속. 신규 서체 금지.
- **공통 재사용**: Nav(current='content')·Footer·SubNav·Modal·Button·Card·Chip·Badge·Reveal·Section·SectionHeader.
- **모션**: 홈 useReveal(th .16)·hover, reduced-motion 대응.
- **반응형**: 확정 브레이크포인트만. 4축 맵→세로, 탐색기 그리드 반응형(3→2→1열), 툴바 접힘, 법정/IP 그리드 축소. 3디바이스 정상.
- **접근성**: 탐색기 탭 `role="tablist"`, 셀렉트 라벨, 카드 키보드(Enter/Space)·포커스, 모달 포커스 트랩·ESC·포커스 복원, 검색 `aria-label`.

## V. 성공 기준 (DoD)
- **구성·인터랙션 = 시안과 일치**(4축·탐색기·상담/상세 모달·스크롤스파이), **비주얼 = 홈 디자인 시스템(Design.md)과 일관**(홈·P1~P3와 나란히 대조). → 시안과 픽셀 1:1 일치는 목표 아님.
- 탐색기 탭·검색·필터·정렬·상세 모달, 상담 모달 검증/성공, 4축 스크롤스파이 동작.
- **신규 색토큰·서체·버튼변형 0**(4축맵/탐색기 --p4 파생 스케일만 예외), `.tint-p4` 적용, `next build` 무오류(홈·P1·P2·P3 회귀 없음).
- 공통 Footer(부정훈련 예방/신고) 정상, 3디바이스 정상, 키보드·reduced-motion 대응.
